## Modos:

O projeto consta com os 3 seguintes modos de jogo:

- PvP (Jogo 1 contra 1)

- Easy (Jogo contra bot simples que escolhe aleatoriamente os movimentos)

- Hard (Jogo contra bot utilizando o algoritmo Minimax)
